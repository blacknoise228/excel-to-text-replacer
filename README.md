# Таблица в Текст

Этот проект позволяет вытащить данные из таблицы и вставить их в шаблон текста вместо знаков вопроса.

## Как работает

1. Вы указываете путь к таблице
2. Вы указываете путь к шаблону
3. Проект вытаскивает данные из таблицы и заменяет знаки вопроса в шаблоне текста на соответствующие значения (необходимо менять структуру в пакете excel и делать корректировки в пакете replacer)
